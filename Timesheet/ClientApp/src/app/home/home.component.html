Hello, world!

This is a timesheet project sample, built with:

  • ASP.NET Core and C# for cross-platform server-side code
  • Angular and TypeScript for client-side code
  • Bootstrap for layout and styling
  • SQL Server with EF Core for database (O/RM)
  • XUnit/FluentAssertions/Moq/Fixture for unit tests

To help you get started:

You can clone the repository here

To run this project, make sure you have all dependencies installed:
  • Visual Studio
  • Sql Server
  • Net Core
  • NodeJS
  • Angular CLI
